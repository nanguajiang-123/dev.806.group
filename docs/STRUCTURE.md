# 806-DEV 技术导航 - 新结构说明

## 项目重构概述

本次重构将原有的博客结构转变为以技术学习导航为核心的知识库，提供系统化的学习路径和实用的技术指南。

## 目录结构

```
content/
├── _index.md                    # 首页 - 技术导航总览
├── general/                     # 通用技术
│   └── _index.md               # 通用技术学习路径
├── frontend/                    # 前端开发
│   └── _index.md               # 前端开发学习路径
├── backend/                     # 后端开发
│   └── _index.md               # 后端开发学习路径
├── gui/                         # GUI开发
│   └── _index.md               # GUI开发学习路径
├── maintaining/                 # 系统维护
│   └── _index.md               # 系统维护学习路径
├── projects/                    # 项目展示
│   └── _index.md               # 项目展示页面
├── about/                       # 关于我们
│   └── _index.md               # 站点介绍和团队信息
└── posts/                       # 原有文章（保留）
    └── user/
```
