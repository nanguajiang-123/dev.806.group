开发工具推荐清单

Author  included in  category 工具推荐
 2024-11-01  2024-11-01  112 words   One minute 

协作与效率

APIFox

https://www.apifox.cn/ - API 设计、开发一体化协作平台。在项目设计的时候需要设计接口文档，可以使用这个工具，方便团队之间协作。

GitHub Actions

https://github.com/features/actions - GitHub 提供的 CI/CD 服务，可以用于自动化构建、测试、部署等。

PaaS 服务

Cloudflare CDN

https://cloudflare.com/ - 免费的 CDN 和 DNS 服务。

Vercel

https://vercel.com/ - 免费的 Node.js 项目部署托管和 CDN 服务。

Framer

https://framer.com/ - 免费的网页设计工具。

前端开发

框架组件库

这里只是放了些符合我审美的组件库

React 生态

  • Mantine - 功能齐全的React组件库
  • MUI - Material-UI，Google设计规范的React实现
  • NextUI - 现代化的React组件库

Vue 生态

  • Vuetify - Material Design的Vue实现

设计资源

  • UIverse - 有很多可以直接使用的 UI 组件
  • Google Fonts - Google 字体库
  • Material Design Icons - Material Design 图标库
  • 阿里巴巴矢量图标库 - 中文图标资源

后端开发

Web 框架

  • FastAPI - Python 的高性能异步 Web 框架
  • Gin - Go 语言的高性能 Web 框架

Web 服务器

  • Caddy - Go 语言编写的 Web 服务器，支持自动部署 SSL 证书，配置清晰易懂，比 Nginx 好用

杂项工具

  • Wappalyzer - Web 技术栈分析器
  • JSON格式化验证 - Json 解析和格式化验证
  • Minecraft - 游戏开发灵感来源
